# VirtualCilia
imitates a real running Cilia in a Unity Scene
Based on FakeCilia

## Needs
* [CiliaUnityPlugin](https://hapticsol.com/s/CiliaUnityPlugin_0274.zip)
* Api Compatibility Level: .NET 4.x (because of SerialPort(System.IO.Ports))
